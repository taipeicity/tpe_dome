資訊釐清-常見問題

Q
為什麼要遠雄停工？
A
  1. 依據建築法58條：建築物在施工中，主管建築機關認有必要時，得隨時加以勘驗， 發現主要構造或位置或高度或面積與核定工程圖樣及說明書不符者，應以書面通知承造人或起造人或監造人，勒令停工或修改
  2. 2015年5月14日，現場勘驗過後發現有高達三分之二樓層的主要構造未按圖施工
  3. 2015年5月20日，市政府負起監督責任，勒令停工

Q
大巨蛋停工後的公安問題，由誰負責？
A
  1. 依據建築法63條，公安問題由承造人遠雄營造及監造人羅興華建築師負起工地安全維護之責任
  2. 停工後，遠雄和羅興華建築師已申請先行報備施工，並且在2015年8月完成大底工程， 經過土木技師公會、結構技師公會、大地技師公會及建築師公會等單位，判讀遠雄所提供資料， 認為基地暫無上浮力問題
  3. 基於工地安全，市政府已於105年4月26日再度邀集專家研討，並要求遠雄提供監測資料，確保無公安問題

Q
市府為何針對大巨蛋提出7項公安基準？
A
  1. 大巨蛋園區量體龐大，超出現行法令，有必要提出7項公安基準降低災害風險
  2. 市府成立安檢小組，考量國際安全標準及案例，歸納出這7項基準
  3. 7項公安基準中，三項由營建署授權臺灣建築中心性能評定審查，四項由臺北市政府依都市計畫定期通盤檢討實施辦法第9條， 及開發行為環境影響評估作業準則第26條審查
  4. 大巨蛋是有史以來規模最大的複合式商業園區，市府有責任保障每位市民的安全

Q
大巨蛋是不是有鏽蝕問題？
A
  1. 依據建築法63條，公安問題由承造人遠雄營造及監造人羅興華建築師負起工地安全維護之責任
  2. 大巨蛋鋼構架如有鏽蝕問題，是承造人遠雄和監造人羅興華建築師應該負起的公安責任
  3. 遠雄營造和羅興華建築師應持續進行維護安全、防範危險的措施，避免發生公安問題
  4. 遠雄應依法維護工地安全，不應推卸責任，市府將持續監督

來源：大巨蛋構架鏽蝕問題，建管處依建築法監督改善

Q
什麼是營造業審議委員會？為什麼他們認為遠雄營造沒有疏失？
A
  1. 營造業審議委員會是由各級主管機關所設，負責審議營造業撤銷或廢止登記事項、營造業獎懲事項及專任工程人員及工地主任處分案件。
  2. 大巨蛋工程，由遠雄巨蛋公司和臺北市政府簽約，遠雄巨蛋公司再發包給遠雄營造公司施工。
  3. 依據營造業法第26條：「營造業承攬工程，應依照工程圖樣及說明書製作工地現場施工製造圖及施工計畫書，負責施工。」規定； 遠雄營造公司表示工程皆依遠雄巨蛋公司提供、建築師簽章之設計圖施工，遠雄營造並不清楚設計圖核准過程。
  4. 但遠雄巨蛋公司及建築師以未經都審、性能審查之設計圖交給承造廠商施工，卻對外聲稱均經建管處核准， 企圖以魚目混珠的方式，掩飾違法施工的事實。
  5. 無法處分承造廠商，是受限於營造業法的漏洞，實際上建築師因未善盡責任監督承造人按「核准」圖施工， 經本市建築師懲戒委員會104年12月24日審議，認確實違反建築師法第18條規定，故予以「停業2年」之懲戒處分。

來源：大巨蛋承造人免議，非遠雄公司免責。已函請中央儘速修正營造業法。

Q
為什麼疏散模擬沒辦法跑到大巨蛋園區外？
A
  1. 人員疏散模擬未進入道路及松菸，是基於災難發生時，週邊環境狀態無法預期、道路要維持消防救災的功能的前提下， 所做的模擬情境條件設定。
  2. 瞬間大量人潮避難時，必須考量週邊環境的安全性，在不造成恐慌情緒下讓人潮安全臨時停留在基地內， 後再有秩序及方向性的疏散方為有效的安全避難模式。
  3. 由於模擬時遠雄綠地規劃尚未完成，無法確定是否能無障礙停留，故模擬時未納入停留範圍。
  4. 完全於基地內檢討逃生避難，是考量因大型活動大量人潮集中時，若發生火災或其他災害(如震災，水災，風災甚至恐怖攻擊事件) 基地內有足夠逃生避難空間，不是指不能離開地界。
